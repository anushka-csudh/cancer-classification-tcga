# Cancer Classification using TCGA Gene Expression Data

Master's Thesis Project - Machine Learning Classification of 5 Cancer Subtypes

## Dataset
TCGA-PANCAN-HiSeq-801x20531 dataset with 801 samples across 5 cancer types

## Models
- Logistic Regression (98.76% accuracy)
- Random Forest
- XGBoost
- Neural Network (GPU-accelerated)
- SVM
- Decision Tree
- Stacked Ensemble

## Results
- Test Accuracy: 98.76%
- Cross-validation Accuracy: 99.84%
- K-means clustering (k=7) revealing molecular subtypes
- SHAP interpretability analysis
